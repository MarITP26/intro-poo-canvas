🎨 Proyecto: Objetos 2D con Canvas y Programación Orientada a Objetos (OOP)
📌 Descripción del proyecto

Este proyecto es una aplicación web desarrollada con HTML, JavaScript y la API Canvas 2D, cuyo objetivo es demostrar el uso de la Programación Orientada a Objetos (POO) para la creación y manipulación de objetos gráficos dentro de un lienzo (canvas).

La aplicación muestra tres ejemplos diferentes de generación de objetos gráficos:

Objeto 2D generado mediante POO

Objeto generado con valores aleatorios

Múltiples objetos generados dinámicamente

Cada uno de estos ejemplos se visualiza en un canvas independiente, permitiendo observar cómo se crean y manipulan los objetos utilizando clases, arreglos y generación de valores aleatorios.

🧠 Objetivo del programa

El objetivo principal del programa es:

Comprender el uso de la API Canvas de HTML5

Aplicar Programación Orientada a Objetos (OOP) en JavaScript

Generar objetos gráficos dinámicamente

Utilizar valores aleatorios para crear figuras diferentes

Dibujar múltiples objetos en un mismo canvas

🧱 Tecnologías utilizadas

El proyecto utiliza las siguientes tecnologías:

HTML5 → estructura de la página

CSS3 → estilos visuales

Bootstrap 5 → diseño moderno y responsivo

JavaScript → lógica del programa

Canvas API 2D → creación y renderizado de gráficos
